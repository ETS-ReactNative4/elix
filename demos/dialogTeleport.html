The dialog below is sitting inside a container that has a CSS transform applied to it, which forces the creation of a stacking context. That would normally clip the dialog inside the container. An appropriate fix for that problem would be to adjust the styling of the container, or move the dialog elsewhere on the page. In cases where that is not possible, however, the dialog's teleportToBodyOnOpen property can be set. The dialog below uses that property to move itself to the document body, thereby breaking the dialog out of its container's stacking context.
